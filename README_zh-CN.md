# AgentSpace Desktop

AgentSpace 是一款 Windows 桌面应用程序，用于裁剪正在运行的后台窗口，并在它们之间提供自动化的剪贴板数据传输。它利用 Win32 API 与 WPF 直接操作原生窗口。

## 功能

- **窗口裁剪**: 在任意窗口上框选区域即可进行局部脱离，裁剪出的内容悬浮在可移动、可缩放的独立 WPF 容器中。
- **原生同步**: 移动或调整容器大小时，底层的原生程序坐标会被同步修改。
- **保留交互**: 裁剪出的应用区域保持完全可交互状态（支持点击、滚动、输入文字）。
- **意图流转**: 自动复制源容器中的选中文字，并通过全屏视觉化追踪、键盘注入技术将其粘贴到目标窗口。
- **DPI 适配**: 支持多显示器及各种缩放比例（如 125%、150%）。

## 使用指南

双击运行 `AgentSpace.exe`。

- `Ctrl + Shift + Space`: 启动选区模式。鼠标左键画框框选部分屏幕，按 `Enter` 确认裁剪。
- **鼠标左键**: 抓住深蓝色的边框拖动容器。
- **调整大小**: 像缩放普通窗口一样拖拽容器边缘，直接调整裁剪范围。
- `Ctrl` + **鼠标滚轮**: 悬停在裁剪出的画面内向下或向上滚动滚轮，可缩放底层原生程序的分辨率比例。
- `ESC`: 关闭当前容器，将被裁剪的窗口物归原主。
- `[ ➦ Route ]`: (或自定义快捷键) 启动全屏黑罩意图流转界面。按 `Tab` 循环切换目标候选人，按 `Delete` 删除不需要的后台程序列表选项，按 `Space` 确认执行复制-传输-粘贴。

## 依赖环境

- 操作系统：Windows 10 / Windows 11
- 框架基座：.NET Core 3.0 (WPF)

## 二次开发

1. 克隆 / Clone 此代码仓库。
2. 使用带有 C# 开发环境的 Visual Studio 或 VS Code 打开 `src/AgentSpace.sln`。
3. 进入 `AgentSpace.App` 目录下执行 `dotnet build` 编译，或 `dotnet run` 调试。
